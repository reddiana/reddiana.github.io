### Base Image

```dockerfile
FROM centos:centos7.6.1810

COPY ./* /mlservice/

RUN yum install -y https://centos7.iuscommunity.org/ius-release.rpm && \
    yum update -y && \
    yum install -y \
         gcc \
         python35u python35u-devel python35u-pip \
         libSM libXrender libXext \
         nginx \
         R-core R-devel && \
    yum clean all && \
    pip3.5 install --upgrade pip && \
    pip3.5 install \
         Flask uWSGI supervisor \
         rpy2

RUN cp -f /mlservice/nginx.conf       /etc/nginx/nginx.conf && \
    cp -f /mlservice/supervisord.conf /etc/supervisord.conf

EXPOSE 80/tcp

CMD ["supervisord"]
```

### 이런걸 더해서~

```bash
pip3 install -r requirements.txt && \
mkdir /usr/share/doc/R-3.5.2/html && \
yum install -y cario cairo-devel libXt-devel && \
Rscript -e "install.packages(c('Cairo', 'ggplot2', 'sqldf', 'png', 'base64enc', 'data.table'), repos='http://cran.biodisk.org/')"
```

### 지저분한거 싹 지울 땐~

```bash
docker rm -f $(docker ps -aq) && \
docker rmi -f $(docker images -f dangling=true -q)
```
